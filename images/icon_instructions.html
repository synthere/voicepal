VoicePal图标生成说明

为了完成Chrome扩展，您需要创建以下尺寸的图标：

  • 16x16 像素
  • 48x48 像素
  • 128x128 像素

SVG 代码

以下是图标的SVG代码，您可以将其保存为icon.svg文件：

生成PNG图标的方法

方法1：使用在线SVG到PNG转换工具

  1. 复制上面的SVG代码
  2. 访问在线SVG到PNG转换工具，如 svgtopng.com 或 convertio.co
  3. 粘贴SVG代码或上传SVG文件
  4. 设置所需的尺寸（16x16、48x48、128x128）
  5. 转换并下载PNG文件
  6. 将文件分别命名为 icon16.png、icon48.png 和 icon128.png
  7. 将这些文件放在 images 文件夹中

方法2：使用图像编辑软件

  1. 在图像编辑软件中打开SVG文件（如Adobe Illustrator、Inkscape等）
  2. 导出为所需尺寸的PNG文件
  3. 将文件分别命名为 icon16.png、icon48.png 和 icon128.png
  4. 将这些文件放在 images 文件夹中

注意：在实际部署扩展之前，您需要替换这些临时图标文件。如果您没有立即可用的图标，可以使用以下简单的彩色方块作为临时图标，直到您有适当的图标设计。

临时图标代码（如果需要）

如果您想快速创建简单的临时图标，可以使用以下HTML代码生成彩色方块图标：

<!DOCTYPE html>
<html>
<head>
  <title>生成临时图标</title>
</head>
<body>
  <canvas id="icon16" width="16" height="16"></canvas>
  <canvas id="icon48" width="48" height="48"></canvas>
  <canvas id="icon128" width="128" height="128"></canvas>
  
  <script>
    // 创建16x16图标
    const ctx16 = document.getElementById('icon16').getContext('2d');
    ctx16.fillStyle = '#6A11CB';
    ctx16.fillRect(0, 0, 16, 16);
    
    // 创建48x48图标
    const ctx48 = document.getElementById('icon48').getContext('2d');
    const gradient48 = ctx48.createLinearGradient(0, 0, 48, 48);
    gradient48.addColorStop(0, '#6A11CB');
    gradient48.addColorStop(1, '#2575FC');
    ctx48.fillStyle = gradient48;
    ctx48.fillRect(0, 0, 48, 48);
    
    // 创建128x128图标
    const ctx128 = document.getElementById('icon128').getContext('2d');
    const gradient128 = ctx128.createLinearGradient(0, 0, 128, 128);
    gradient128.addColorStop(0, '#6A11CB');
    gradient128.addColorStop(1, '#2575FC');
    ctx128.fillStyle = gradient128;
    ctx128.fillRect(0, 0, 128, 128);
    
    // 添加文本
    ctx48.fillStyle = 'white';
    ctx48.font = 'bold 24px Arial';
    ctx48.textAlign = 'center';
    ctx48.textBaseline = 'middle';
    ctx48.fillText('VP', 24, 24);
    
    ctx128.fillStyle = 'white';
    ctx128.font = 'bold 64px Arial';
    ctx128.textAlign = 'center';
    ctx128.textBaseline = 'middle';
    ctx128.fillText('VP', 64, 64);
    
    // 右键点击画布并选择"另存为图片"来保存图标
  </script>
</body>
</html>
